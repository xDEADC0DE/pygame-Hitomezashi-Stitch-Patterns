# Hitomezashi Stitch Patterns
2<sup>e</sup> projet avec Pygame

Génère des motifs intéressants en fonction des lettres tapées. Appuyez sur `Entrée` pour passer du mode «&nbsp;ligne&nbsp;» au mode «&nbsp;colonne&nbsp;»
